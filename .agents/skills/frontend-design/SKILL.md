# Frontend Design Skill

Esta skill define las reglas estrictas de diseño UI/UX moderno para el proyecto, evadiendo la estética genérica producida por IA.

## Reglas Principales:
1. **Tipografía:** Utilizar fuentes sans-serif modernas (como Inter, Space Grotesk) con una jerarquía clara. Evitar tamaños de fuente estándar sin pensar en la proporción.
2. **Espaciado y Layout:** Usar un sistema de grilla de 8px. Dar suficiente espacio de respiración (whitespace) a los elementos.
3. **Color:** No usar colores primarios saturados por defecto (ej. azul puro o rojo puro). Usar paletas cuidadosamente seleccionadas (ej. Navy Deep, Cyan oscuro) con matices sutiles.
4. **Bordes y Sombras:** Usar sombras suaves y difusas (`box-shadow: 0 12px 32px -18px rgba(0,0,0,0.5)`) en lugar de bordes duros. Implementar sutiles efectos de glassmorphism donde tenga sentido.
5. **Autenticidad:** Los diseños deben verse premium, asimétricos cuando sea necesario, y altamente responsivos. Evitar layouts "boxy" o componentes centrados genéricos que gritan "generado por IA".
